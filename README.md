# Pengakap Attendance System Documentation

## Introduction
The Pengakap Attendance System is designed to manage and track the attendance of pengakap members effectively. This system allows for easy recording, retrieval, and analysis of attendance data.

## Features
- **Member Registration:** Allow new members to register into the system.
- **Attendance Marking:** Enable leaders to mark attendance during meetings and events.
- **Reports:** Generate reports on attendance for individual members or groups.
- **Notification System:** Notify members about upcoming events and meetings.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shairazi-91/sistem-kehadiran-pengakap.git
   ```
2. Navigate to the directory:
   ```bash
   cd sistem-kehadiran-pengakap
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To run the application, use the following command:
```bash
npm start
```
